# Team 01 Project

## How to Run Locally

### Prerequisites
- Node.js (v14 or higher)
- MongoDB

### Backend Setup
```bash
cd backend
npm install
npm start
```

### Frontend Setup
```bash
cd frontend
npm install
npm run dev
```

### Environment Variables
Create `.env` file in backend directory:
```
MONGODB_URI=mongodb://localhost:27017/team01
PORT=5000
```

### Access
- Frontend: http://localhost:5173
- Backend: http://localhost:5000
